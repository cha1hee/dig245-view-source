how to open safari source code

  1. open safari
  2. in the top left of the menu bar, hover over safari
  3. click on preference
  4. go to 'advanced' tab
  5. check box in bottom of the tab that says 'Show Develop menu in menu bar'
  6. close out of the preferences window
  7. now to view source code of any webpage, press option+command+u

other ways to use DevTools

  • Monitoring the performance of the webpage based off of runtime of activity.
  • Find color hex values on the webpage, and color palette used.
  • Edit elements on page to test how certain edits in element would look.
